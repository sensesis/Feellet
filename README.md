# ✨App Description
사용자가 소비 당시의 감정을 함께 기록함으로써 자신의 소비 습관을 되돌아볼 수 있도록 돕는 감정 중심 가계부 앱  
<br>
소비를 통해 느낀 감정(후회, 만족 등)을 함께 기록함으로써, 자신의 소비 성향과 심리를 스스로 이해하고 현명한 소비 습관을 형성하게 함.  
회원가입 및 로그인 기능과 함께, 가계부 작성, 삭제, 수정, 감정 선택, 대시보드 등 다양한 기능 제공
